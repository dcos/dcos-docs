# Description
Remove a package repository from DC/OS.

# Usage

```bash
dcos package repo remove <repo-name> [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<repo-name>`   |             |  Name of the package repository. For example, `Universe`. |

# Positional arguments
