# Description
Cancel the in-progress deployment of an application.

# Usage

```bash
dcos marathon deployment stop <deployment-id> [OPTION]
```

# Options

None.

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<deployment-id>`   |             |  The deployment ID. |

# Parent command

| Command | Description |
|---------|-------------|
| [dcos marathon](/docs/1.11/cli/command-reference/dcos-marathon/) | Deploy and manage applications to DC/OS. |
